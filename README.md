# Storage-DataStructure
commonly used data structures in storage system
## 1. SkipList
`ref: https://lotabout.me/2018/skip-list/`
SkipList是基于单项链表建立的，每个节点上面建立多层索引，跳跃式查询目标值，达到复杂度`O(logn)`的插入和删除复杂度
